# MIT Applied Data Science Program

A collection of projects and coursework completed during MIT's 12-week Applied Data Science Program (ADSP), covering the full data science pipeline from foundations and statistical analysis through deep learning and recommendation systems.

## Program Curriculum

| Module | Topics |
|--------|--------|
| Foundations of Data Science | Statistics, probability, data wrangling |
| Data Analysis & Visualization | Exploratory analysis, visual storytelling |
| Machine Learning | Regression, regularization, bootstrapping, model evaluation, classification |
| Practical Data Science | Decision trees, random forests, time series |
| Deep Learning | Artificial neural networks, CNNs, graph neural networks |
| Recommender Systems | Collaborative and content-based filtering |
| Self-Study Electives | Big data analytics, AI on cloud |

## Projects

### Street View House Number (SVHN) Digit Recognition

Built and compared ANN and CNN architectures to recognize digits in Google's Street View House Numbers dataset. Explored how convolutional layers improve performance over fully connected networks on image data.

**Tech:** Python, TensorFlow/Keras, NumPy, Matplotlib

### Facial Emotion Recognition *(Capstone)*

Developed a CNN from scratch for classifying facial emotions, then benchmarked it against pretrained transfer learning architectures. Analyzed trade-offs between model complexity, training efficiency, and classification accuracy.

**Tech:** Python, TensorFlow/Keras, Transfer Learning (VGG, ResNet, etc.), NumPy, Matplotlib
